🛒 Cartify – React E-Commerce Website

Cartify is a modern React.js e-commerce web application built to simulate a real online shopping experience. It includes product browsing, cart management, payment simulation, and order tracking features.

🚀 Features
🛍️ Product listing with categories
🔍 Search and filter products
🛒 Add to cart & quantity management
❤️ Wishlist feature
🌙 Dark mode support
💳 Payment page (Debit Card, Credit Card, Net Banking, Wallet, UPI, COD)
📱 UPI QR code payment simulation
✅ Order confirmation page
🚚 Order tracking system
🔐 Login button UI (frontend only)
🎨 Responsive and modern UI

🧑‍💻 Tech Stack
React.js
JavaScript (ES6+)
React Hooks (useState)
